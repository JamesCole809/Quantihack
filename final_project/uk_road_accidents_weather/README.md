# UK Road Accidents x Weather

This folder is the dedicated workspace for the final research project built around the UK road accidents dataset and the weather dataset.

The raw source files live under `data/raw/` as links back to the repo's chosen source datasets, so this project folder stays self-contained without duplicating the large parquet files again.

## Structure

- `data/raw/`: linked source datasets used by the project
- `data/processed/`: merged daily and monthly analysis tables generated by the script
- `outputs/`: figures and summary metrics generated from the analysis
- `scripts/build_analysis.py`: reproducible project build script
- `REPORT.md`: generated write-up of the main findings

## How To Rebuild

Run:

```bash
python3 final_project/uk_road_accidents_weather/scripts/build_analysis.py
```

That script rebuilds the processed tables, rewrites the dashboard figure, and refreshes `REPORT.md` from the latest metrics.

## Analysis Frame

The analysis uses the shared **2009-2020** overlap window:

- Road accidents: legacy accident file for 2009-2015 plus the recent accident file for 2016-2020
- Weather: daily national weather aggregates derived from the selected weather parquet

Because the weather source in this repo does not provide a clean station-to-accident spatial key, the project is designed as a **daily national-level relationship study** rather than a location-by-location merge.
